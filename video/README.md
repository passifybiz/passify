# Passify Product Demo Video

A 60-second animated product demo built with [Remotion](https://remotion.dev).

## Scenes

1. **Intro** (0-5s) — Logo animation with tagline
2. **Problem** (5-12s) — Three pain points animated in
3. **Solution** (12-20s) — Three pillars with spring animation
4. **Dashboard** (20-30s) — Live dashboard mockup with data appearing
5. **API Integration** (30-40s) — Code request/response animation
6. **Compliance** (40-50s) — Rule cards animated
7. **CTA** (50-60s) — Call to action with scale spring

## Usage

```bash
cd video
npm install
npm run dev      # Open Remotion Studio to preview
npm run render   # Render to MP4
```

## Requirements

- Node.js 18+
- Chrome/Chromium (for rendering)
